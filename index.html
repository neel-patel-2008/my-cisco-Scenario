Quality of Service (QoS)

  • Traffic Classes:
    • Voice Traffic (DSCP EF): 30% bandwidth guarantee
    • Business Critical (DSCP AF31): 25% bandwidth
    • Standard Business (DSCP AF21): 35% bandwidth
    • Best Effort: 10% bandwidth
  • QoS Policy Application:
    • Applied on: All router WAN interfaces
    • Direction: Both inbound and outbound
    • Queue management: CBWFQ with LLQ for voice

Implementation Phases

  • Phase 1: Basic Infrastructure:
    • Place and configure core routers and switches
    • Configure basic VLANs and trunk links
    • Set up inter-VLAN routing
    • Test basic connectivity within each site
  • Phase 2: WAN Connectivity:
    • Configure serial interfaces for WAN links
    • Implement OSPF routing protocol
    • Test site-to-site connectivity
    • Configure internet access and NAT
  • Phase 3: Network Services:
    • Configure DHCP services
    • Set up DNS servers
    • Configure wireless infrastructure
    • Test all network services
  • Phase 4: Security Implementation:
    • Configure ACLs for traffic filtering
    • Implement port security
    • Set up VPN tunnels
    • Configure firewall rules
  • Phase 5: Advanced Features:
    • Implement QoS policies
    • Configure network redundancy (HSRP, STP)
    • Set up network monitoring
    • Performance optimization

    • show ip interface brief
    • show ip route
    • ping [destination]
    • traceroute [destination]
  • VLAN Troubleshooting:
    • show vlan brief
    • show interfaces trunk
    • show interfaces switchport
  • OSPF Troubleshooting:
    • show ip ospf neighbor
    • show ip ospf database
    • show ip ospf interface
  • VPN Troubleshooting:
    • show crypto isakmp sa
    • show crypto ipsec sa
    • show crypto map
